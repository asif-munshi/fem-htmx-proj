# FEM-HTMX-PROJ

## Getting Started

### Via `go install` (Recommended)

With go 1.22 or higher:

```bash
go install github.com/cosmtrek/air@latest
```

Start the server

```bash
air
```
Open [http://localhost:42069](http://localhost:42069) with your browser to see the result.
